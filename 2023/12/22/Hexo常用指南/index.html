Hexo常用命令

1、新建特殊页面

hexo new page categories
特殊页 页面 设置
categories 分类页面 type: categories
tags 标签页面 type: tags
archives 归档页面 type: archives
about 关于页面 type: about
links 友链页面 type: links

2、本地预览

hexo serve

3、更改端口

hexo server -p 5000

4、清除缓存文件

hexo clean

5、部署

hexo g -d

Hexo自定义

1、解析标题标签插入目录

  1. 新增 toc.ejs 文件
<% if (page.toc != false) { %>
  <div id="toc">
      <strong class="sidebar-title">目录</strong>
      <%- toc(page.content, {list_number : false}) %>
  </div>
<% } %>
  • article.ejs
+       <%- partial('toc') %>
        <%- page.content %>
  1. 修改_config.yml文件
toc:
  maxdepth: 3
  class: toc
  slugify: transliteration
  decodeEntities: false
  anchor:
    position: after
    symbol: '####'
    style: header-anchor
  1. 根据需要修改css样式
/* 自定义添加toc样式 */
#main {
    position: relative;
}
#main #toc {
    position: fixed;
    top: 100px;
    right: 20px;
    background-color: #eee;
    padding-right: 10px;
    max-height: 540px;
    font-size: 0.8em;
    color: color-link;
    overflow-y: auto;
}
#main #toc .sidebar-title {
    margin: 20px;
}
/* #main #toc .toc-link, #main #toc .toc-child {
    list-style: none;
} */
#main #toc .toc .toc-item a span {
    display: block;
    width: 140px;
    flex-wrap: nowrap;
    text-overflow: ellipsis;
    overflow: hidden;
}
