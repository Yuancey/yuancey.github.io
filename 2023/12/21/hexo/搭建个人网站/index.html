HEXO

1、Install

npm install -g hexo-cli

More info: Hexo文档

2、Init

hexo init (你的文件夹名称)
cd (你的文件夹名称)
npm install

3、配置_config.yml

参数 描述
title 网站标题
subTitle 网站副标题
description 网站关键词
author 作者
language 网站语言
timezone 网站时区
url 网址

Github配置

1、生产SSH密钥

 ssh-keygen -t rsa -C "（你的Github注册邮箱）" #（你的Github注册邮箱）不要加

1.1、根据提示找到.ssh文件夹
1.2、用记事本或其他文本编辑器打开id_rsa.pub并复制其内容

2、New SSH key

Settings—>SSH and GPG keys—>New SSH key

3、new repository

  • 命名：（你的Github用户名）.github.io
  • 复制SSH地址（git@github.com:开头）

4、配置_config.yml

deploy:
  type: git
  repo: （你的SSH地址） #“（你的SSH地址）”不要加“（）”
  branch: main

5、安装

hexo-deployer-git

$ npm install hexo-deployer-git --save

6、部署

$ hexo clean
$ hexo g -d
