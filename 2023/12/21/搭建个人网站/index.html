HEXO搭建个人网站

2023/12/21

Welcome to my very first bolg ! [Hexo] If you get any problems when using Hexo, you can find the answer in troubleshooting.

HEXO使用

Install

$ npm install -g hexo-cli

More info: Hexo文档

Init

$ hexo init (你的文件夹名称)
$ cd (你的文件夹名称)
$ npm install

配置_config.yml

参数 描述
title 网站标题
subTitle 网站副标题
description 网站关键词
author 作者
language 网站语言
timezone 网站时区
url 网址

modify the theme

参考博客
仓库地址

Github配置

生产SSH密钥

$ ssh-keygen -t rsa -C "（你的Github注册邮箱）" #（你的Github注册邮箱）不要加

1、根据提示找到.ssh文件夹
2、用记事本或其他文本编辑器打开id_rsa.pub并复制其内容

New SSH key

Settings—>SSH and GPG keys—>New SSH key

new repository

  • 命名：（你的Github用户名）.github.io
  • 复制SSH地址（git@github.com:开头）

配置_config.yml

deploy:
  type: git
  repo: （你的SSH地址） #“（你的SSH地址）”不要加“（）”
  branch: main

其他

hexo-deployer-git

$ npm install hexo-deployer-git --save

部署

$ hexo clean
$ hexo g -d
