3、使用PicGo-Core（Command line）

–> typora>文件>偏好设置>图像–选择PicGo-Core–点击[下载更新]–打开配置文件

{
  "picBed": {
    "uploader": "aliyun",
    "aliyun": {
    "accessKeyId": "", //ram的accessKey
    "accessKeySecret": "", //ram的accessKey
    "bucket": "", // 桶名
    "area": "", // 地域节点
    "path": "img/", // 存储路径
     "customUrl": "", // Bucket域名，注意要加 http://或者 https://
     "options": "" // 针对图片的一些后缀处理参数 PicGo 2.2.0+ PicGo-Core 1.4.0+
    }
  },
  "picgoPlugins": {}
}
